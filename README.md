# lidar-assessment

A clean, modern single-page business assessment web app.

## Features

- 5 assessment sections:
  - Governance
  - Operations
  - Customer Experience
  - Quality
  - Technology
- 3 questions per section with a 1–5 rating scale.
- Automatic calculation of:
  - Section-level scores
  - Overall score
  - Maturity classification (`Emerging`, `Developing`, `Mature`)
- Dynamic recommendations for sections with low scores.

## Run locally

Open `index.html` in your browser.
